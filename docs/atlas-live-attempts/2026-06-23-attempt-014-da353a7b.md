### Key Finding

The rejection ratio (72-74%) appears to be a **systemic property of the web search results**, not the curation thresholds. Across 3 attempts with different code and prompts, the ratio stays remarkably stable at ~72-74%. Lowering thresholds had negligible effect because the rejected sources are genuinely low-quality — not borderline cases that thresholds would rescue.

Future rejection ratio improvements may need a fundamentally different approach:
- Broader initial search (more results per query)
- Different search engine configuration
- More nuanced quality scoring that goes beyond snippet/title length
- Source preprocessing or enrichment before scoring
